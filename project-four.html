Project Four — Fruit Ninja

A recreation of the classic 2010 mobile game "Fruit Ninja" by Halfbrick made on Java Processing.

Project Overview

Fruit Ninja is a classic mobile game involving slicing fruit that is thrown onto the screen while avoiding bombs. The 'run' can also be ended once a certain amount of lives are depleted as a result of too many fruits being missed by the user. This project aims to recreate these objectives and mechanics as closely as possible to the original game using Java Processing. As an add on, the player can also enter Zen mode, where there are no bombs and instead replaced with the frenzy fruits that can be spawned when pressing 1 and 2 on the keyboard.

Live Demo

Key Features

  • Dragging the mouse creates a slicing effect. Sliced fruits realistically split into two pieces and fall off-screen, leaving behind a colorful splash stain on the background.
  • Rewards skillful play by tracking a high score for each mode. Slicing multiple fruits in quick succession triggers a combo counter, awarding significant bonus points.
  • The game is built with a clean, object-oriented structure where fruits, buttons, stains, and combos are managed as individual objects with their own properties and behaviors, such as simulated physics for fruit trajectory.
