Project Two - STM32 Environmental Monitor

Interfacing the DHT11 temperature and humidity sensor with an STM32 microcontroller using HAL libraries.

Project Overview

In this project, I crafted an embedded system in C that utilizes the STM32 HAL library to interface with a DHT11 sensor, allowing for the precise measurement of temperature and humidity. The project features time-critical communication and displays real-time data on an integrated LCD screen via the I2C communication protocol, serving as a functional environmental monitoring system. However, the sensor does have a slight error margin of ±2°C for temperature and ±5% for humidity, which is acceptable for general monitoring purposes.

Live Demo (coming soon!)

Key Features

  • The core of this project is built using the STM32 HAL (Hardware Abstraction Layer) libraries, which simplifies the communication and control of the microcontroller.
  • The system utilizes STM32 timers to create microsecond delays, which are critical for the time-sensitive communication protocol of the DHT11 sensor.
  • The project is configured to read a 40-bit data packet from the sensor, which includes humidity and temperature data, and a checksum for data validation.
