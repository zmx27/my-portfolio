Project Three — Flappy Electron

A Physics simulation where the movement of the "electron" is affected by the user-controlled magnitude and direction of the surrounding electric and magnetic fields.

Project Overview

This project features an obstacle course that was inspired by the popular mobile game Flappy Bird. The goal of the game is to get an electron through several openings in the obstacles by changing the strength and direction of the magnetic and electric fields. These are controlled by the user, so they must carefully control the electron and get it to safely pass through all the obstacles and reach the end.

Live Demo

Key Features

  • User Input: The ‘wasd’ keys are used to control the magnetic field, while the arrow keys are used to control the electric field.
  • Subsequent levels will only be unlocked when prior levels are beaten.
  • The 'Reset', 'Pause', and 'Return to Home' buttons make for easy maneuverability.
